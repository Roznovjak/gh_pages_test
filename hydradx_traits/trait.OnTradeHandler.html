fn on_trade_weight() -> Weight

Known overhead for a trade in on_initialize/on_finalize. Needs to be specified here if we don’t want to make AMM pools tightly coupled with the price oracle pallet, otherwise we can’t access the weight. Add this weight to an extrinsic from which you call on_trade.
